MINECRAFT JAVA EDITION 1.18.2

Selasa, 22 Februari 2022


Update Minecraft Java Edition Versi 1.18.2 telah dikonfirmasi, Kami sekarang merilis 1.18.2 untuk Minecraft: Java Edition. Rilis ini menambahkan tag universal dan perintah placefeature. Ini juga memperkenalkan kemungkinan paket data eksperimental untuk menyesuaikan pembuatan gua dan menambahkan struktur khusus baru ke dunia.
