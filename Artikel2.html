MINECRAFT JAVA EDITION 1.18.2

Selasa, 22 Februari 2022


Update Minecraft Java Edition Versi 1.18.2 telah dikonfirmasi, Kami sekarang merilis 1.18.2 untuk Minecraft: Java Edition. Rilis ini menambahkan tag universal dan perintah placefeature. Ini juga memperkenalkan kemungkinan paket data eksperimental untuk menyesuaikan pembuatan gua dan menambahkan struktur khusus baru ke dunia.

Selain itu, versi ini berisi pembaruan kepatuhan untuk pemain Korea Selatan.

Perubahan di 1.18.2

  • Pengguna yang menjalankan Minecraft di lingkungan 32-bit akan menerima peringatan di menu utama tentang akhir yang akan datang dari dukungan lingkungan 32-bit
  • Jika Anda bermain di Korea Selatan, kami menambahkan timer dan pemberitahuan gameplay sesuai dengan peraturan game untuk membantu mengingatkan pemain agar sesekali beristirahat dari gameplay
  • Sebuah seed dari angka nol tidak lagi ditangani sebagai kasus khusus

Perubahan Teknis di 1.18.2

  • Perintah tambahan: placefeature
  • Perintah locate sekarang menggunakan parameter id namespaced
  • Parameter perintah locate sekarang menjadi struktur yang dikonfigurasi daripada tipe struktur. Misalnya, sekarang Anda dapat menggunakan /locate village_desert atau /locate shipwreck_beached

PLACEFEATURE Command

Perintah baru yang menempatkan fitur yang dikonfigurasi di lokasi tertentu. Sintaks: fitur tempat [pos] Parameter:

  • id: ID dengan spasi nama dari fitur yang dikonfigurasi untuk ditempatkan
  • pos: Posisi yang akan digunakan sebagai asal penempatan fitur (jika dihilangkan, ~ ~ ~ digunakan)

CONFIGURABLE CAVES

  • Registri baru telah ditambahkan untuk fungsi Density (gua dibuat dengan menggabungkannya bersama-sama)
  • Pengaturan kebisingan mendapat bidang baru noise_router (dan kehilangan beberapa bendera), lihat folder worldgen/noise_settings di laporan worldgen
  • Pengaturan kebisingan menghubungkan bagian-bagian yang digerakkan oleh data dari generasi gua dengan sisa kode

CONFIGURED STRUCTURES AND STRUCTURE SETS

  • Paket data eksperimental dapat menambahkan kumpulan struktur baru
  • Bidang fitur di predikat lokasi sekarang merujuk pada fitur yang dikonfigurasi
  • Bidang tujuan fungsi tabel jarahan explore_map sekarang menjadi id tag fitur yang dikonfigurasi
  • Fungsi tabel jarahan explore_map tidak lagi secara otomatis menetapkan nama tampilan peta
